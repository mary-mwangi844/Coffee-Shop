# Maya's Coffee Shop - Next.js

This is a Next.js version of the Maya's Coffee Shop website, converted from the original HTML file.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```

## Project Structure

- `app/page.tsx` - Main page component with the coffee shop content
- `app/layout.tsx` - Root layout with metadata
- `app/globals.css` - Global styles extracted from the original HTML

## Features

- Responsive design
- Smooth scrolling navigation
- Animated coffee cup with steam effects
- Coffee menu with pricing
- Equipment showcase
- Gifts & snacks section
- About section with statistics
- Beautiful CSS animations
